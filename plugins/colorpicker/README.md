# Don Rowlett Color Picker

- 上游: [ColorPicker](https://sourceforge.net/projects/npp-plugins/files/ColorPicker/) v2.3 (x86)
- 作者: Don Rowlett
- 形态: 完整实现（CLI 包装）

Select and convert color codes in various formats.

## 命令

- `hex-to-rgb` — Hex to RGB  (ok)
- `rgb-to-hex` — RGB to Hex  (ok)

## 上游原版本

v2.3（本 macOS 实现为 v1.0.0）
