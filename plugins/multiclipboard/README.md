# MultiClipboard

- 上游: [MultiClipboard](https://sourceforge.net/projects/npp-plugins/files/MultiClipboard/) v2.1.0.0 (x86)
- 作者: LoonyChewy
- 形态: stub 占位

Multiple text buffers via copying/cutting.

## 命令

- `show-history` — Show Clipboard History  (stub)
- `paste-buffer` — Paste from Buffer  (stub)

> 本插件当前为 stub，真正可用需后续原生 Swift UI 实现（clipboard history panel）。
