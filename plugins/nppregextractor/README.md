# NppRegExTractor

- 上游: [NppRegExTractorPlugin](https://github.com/viper3400/RegExTractor/wiki/de_userdocumentation) v2.1.0 (x64)
- 作者: Jan Graefe
- 形态: 完整实现（CLI 包装）

Search regular expressions and extract matches.

## 命令

- `extract-emails` — Extract Email Addresses  (ok)
- `extract-urls` — Extract URLs  (ok)
- `extract-ips` — Extract IP Addresses  (ok)

v2.1.0（本 macOS 实现为 v1.0.0）
