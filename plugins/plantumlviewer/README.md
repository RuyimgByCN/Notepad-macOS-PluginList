# PlantUML Viewer

- 上游: [PlantUmlViewer](https://github.com/Fruchtzwerg94/PlantUmlViewer) v1.9.0.13 (x64)
- 作者: Philipp Schmidt
- 形态: 完整实现（CLI 包装）

Generate, view and export PlantUML diagrams.

## 命令

- `extract-diagrams` — Extract PlantUML Blocks  (ok)
- `validate-syntax` — Validate PlantUML Syntax  (ok)
- `render-to-file` — Render Diagram to PNG  (stub)

## 外部依赖

- `plantuml`: brew install plantuml
