# Preview HTML

- 上游: [PreviewHTML](https://github.com/rdipardo/npp_preview) v1.4.5.0 (x64)
- 作者: Martijn Coppoolse
- 形态: stub 占位

Preview HTML files inside an embedded browser.

## 命令

- `validate-html` — Validate HTML Tags  (ok)
- `preview` — Preview HTML  (stub)

> 本插件当前为 stub，真正可用需后续原生 Swift UI 实现（HTML preview (WKWebView)）。
