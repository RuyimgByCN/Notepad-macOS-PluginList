# XML Tools

- 上游: [XMLTools](https://github.com/morbac/xmltools) v3.1.1.13 (x64)
- 作者: Nicolas Crittin
- 形态: 完整实现（CLI 包装）

Small set of useful tools for editing XML.

## 命令

- `format-xml` — Format XML  (ok)
- `validate-xml` — Validate XML  (ok)
- `escape-xml` — Escape XML Special Characters  (ok)

## 外部依赖

- `xmllint`: preinstalled on macOS (libxml2)

## 上游原版本

v3.1.1.13（本 macOS 实现为 v1.0.0）
