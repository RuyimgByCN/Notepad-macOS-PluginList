# DSpellCheck

- 上游: [DSpellCheck](https://github.com/Predelnik/DSpellCheck) v1.5.0 (x64)
- 作者: Sergey Semushin
- 形态: 完整实现（CLI 包装）

Spell-checking with underlining of spelling mistakes.

## 命令

- `list-misspellings` — List Misspelled Words  (ok)
- `count-errors` — Count Misspelled Words  (ok)

## 外部依赖

- `hunspell`: brew install hunspell

## 上游原版本

v1.5.0（本 macOS 实现为 v1.0.0）
