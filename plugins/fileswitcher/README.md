# File Switcher

- 上游: [FileSwitcher](https://github.com/bruderstein/FileSwitcher) v1.0.3.0 (x86)
- 作者: Dave Brotherstone
- 形态: stub 占位

Switch the active buffer using just the keyboard.

## 命令

- `list-buffers` — List Open Buffers  (stub)
- `switch-buffer` — Switch Buffer by Name  (stub)

> 本插件当前为 stub，真正可用需后续原生 Swift UI 实现（keyboard buffer switcher panel）。

v1.0.3.0（本 macOS 实现为 v1.0.0）
