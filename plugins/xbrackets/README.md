# XBrackets Lite

- 上游: [XBrackets](https://github.com/d0vgan/npp-XBracketsLite) v1.3.1 (x64)
- 作者: Vitaliy Dovgan
- 形态: 完整实现（CLI 包装）

Autocompletion of brackets.

## 命令

- `wrap-paren` — Wrap with Parentheses  (ok)
- `wrap-bracket` — Wrap with Brackets  (ok)
- `wrap-brace` — Wrap with Braces  (ok)
- `wrap-quote` — Wrap with Double Quotes  (ok)

## 上游原版本

v1.3.1（本 macOS 实现为 v1.0.0）
