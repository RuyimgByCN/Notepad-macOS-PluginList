# NppExec

- 上游: [NppExec](https://github.com/d0vgan/nppexec) v0.8.10 (x64)
- 作者: Vitaliy Dovgan
- 形态: 完整实现（CLI 包装）

Execute commands or saved scripts without leaving Notepad++.

## 命令

- `exec-command` — Execute Selection as Command  (ok)
- `echo-selection` — Echo Selection  (ok)

## 上游原版本

v0.8.10（本 macOS 实现为 v1.0.0）
