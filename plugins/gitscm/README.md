# GitSCM

- 上游: [GitSCM](https://github.com/vinsworldcom/nppGitSCM) v1.4.10.1 (x64)
- 作者: Michael J. Vincent
- 形态: stub 占位

GUI for already installed Git SCM.

## 命令

- `git-status` — Git Status  (ok)
- `git-branch` — Git Branch  (ok)
- `git-log` — Git Log (last 10)  (ok)
- `full-gui` — Open Full Git GUI  (stub)

## 外部依赖

- `git`: install from https://git-scm.com

> 本插件当前为 stub，真正可用需后续原生 Swift UI 实现（full Git GUI panel）。
