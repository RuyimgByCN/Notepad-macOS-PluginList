# Explorer

- 上游: [Explorer](https://github.com/oviradoi/npp-explorer-plugin) v1.9.9.0 (x64)
- 作者: Jens Lorenz

File browser.

## 命令

- `list-directory` — List Current Directory  (ok)
- `current-path` — Show Current Path  (ok)
- `browse-tree` — Browse Folder Tree  (stub)

> 本插件当前为 stub，真正可用需后续原生 Swift UI 实现（file browser side panel (NSOutlineView)）。
