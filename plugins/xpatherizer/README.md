# XPatherizerNPP

- 上游: [XPatherizerNPP](https://code.google.com/p/xpatherizernpp/) v2.10 (x86)
- 作者: bguenthner
- 形态: 完整实现（CLI 包装）

Analyze multiple XPath queries with reverse lookup.

## 命令

- `list-elements` — List XML Elements  (ok)
- `count-elements` — Count XML Elements  (ok)
- `xpath-query` — Run XPath Query  (stub)
