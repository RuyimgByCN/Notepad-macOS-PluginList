# DoxyIt

- 上游: [DoxyIt](https://github.com/dail8859/DoxyIt) v0.4.4 (x64)
- 作者: Justin Dailey
- 形态: stub 占位

Supports creating Doxygen comments.

## 命令

- `generate-comment` — Generate Doxygen Comment  (ok)
- `convert-javadoc` — Convert to Javadoc Style  (ok)

> 本插件当前为 stub，真正可用需后续原生 Swift UI 实现（interactive Doxygen comment editor integration）。
