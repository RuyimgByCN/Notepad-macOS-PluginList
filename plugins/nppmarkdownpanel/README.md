# Markdown Panel

- 上游: [NppMarkdownPanel](https://github.com/mohzy83/NppMarkdownPanel) v0.9.1 (x64)
- 作者: Mohzy83
- 形态: stub 占位

Lightweight plugin to display rendered Markdown files.

## 命令

- `extract-markdown` — Extract Markdown Content  (ok)
- `render-panel` — Render Markdown Panel  (stub)

> 本插件当前为 stub，真正可用需后续原生 Swift UI 实现（Markdown panel (WKWebView)）。

v0.9.1（本 macOS 实现为 v1.0.0）
