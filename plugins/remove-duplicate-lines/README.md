# Remove Duplicate Lines

- 上游: [Remove Duplicate Lines](https://github.com/gurikbal/Remove_dup_lines) v1.3.0.0 (x64)
- 作者: G. Singh
- 形态: 完整实现（CLI 包装）

Remove duplicate lines without removing empty lines.

## 命令

- `remove-dups` — Remove Duplicate Lines  (ok)
- `sort-unique` — Sort and Unique  (ok)
- `count-duplicates` — Count Duplicate Lines  (ok)
