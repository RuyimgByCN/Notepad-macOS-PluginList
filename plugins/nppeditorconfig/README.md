# EditorConfig

- 上游: [NppEditorConfig](https://github.com/editorconfig/editorconfig-notepad-plus-plus) v0.4.0 (x64)
- 作者: EditorConfig Team
- 形态: 完整实现（CLI 包装）

EditorConfig defines coding conventions for a project.

## 命令

- `parse-editorconfig` — Parse .editorconfig  (ok)
- `show-indent` — Show Indent Style  (ok)
