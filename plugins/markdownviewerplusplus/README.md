# MarkdownViewer++

- 上游: [MarkdownViewerPlusPlus](https://nea.github.io/MarkdownViewerPlusPlus/) v0.8.2 (x64)
- 作者: nea
- 形态: stub 占位

View Markdown/CommonMark rendered on-the-fly.

## 命令

- `extract-markdown` — Extract Markdown Content  (ok)
- `render-preview` — Render Markdown Preview  (stub)

> 本插件当前为 stub，真正可用需后续原生 Swift UI 实现（live Markdown render (WKWebView)）。
