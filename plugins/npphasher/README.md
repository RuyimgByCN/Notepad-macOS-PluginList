# NppHash

- 上游: [NppHasher](https://github.com/npp-plugins/hasher) v1.0 (x86)
- 作者: Don HO
- 形态: 完整实现（CLI 包装）

Computes the hash of selected text.

## 命令

- `md5` — MD5 Hash  (ok)
- `sha256` — SHA-256 Hash  (ok)
- `base64-encode` — Base64 Encode  (ok)
- `base64-decode` — Base64 Decode  (ok)
